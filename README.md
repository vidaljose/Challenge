# Challenge
Development Challenge for Node.js This challenge aims to evaluate basic skills in Node.js development, and a bit of data/entity modeling. The idea is to build an HTTP REST API.

## start
```
npm install 
```
## Endpoints

### jwt key
```
GET ':/key'
```
### movie
```
GET ':/'
GET '/limit/{ number }'
GET '/sorted'
GET '/sorted/{ number }'
POST '/' 
```
### others
```
POST '/actor'
POST '/director'
```
### tvshow
```
GET '/{ string }' 
POST '/'
```
